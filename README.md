## 📈 Improving Performance

If the software feels slow when you view large RAW files, consider these tips.

* Close other programs: Free up your system memory by closing background browsers or applications.
* Use a fast drive: Keep your image library on a solid-state drive rather than an older mechanical drive.
* Hardware Acceleration: Ensure you enabled hardware acceleration under the options menu in the settings tab. This uses your graphics card to speed up image rendering.
